# 🎮 High Game Priority

[![Windows](https://img.shields.io/badge/Platform-Windows-0078D6?style=flat-square&logo=windows)](https://www.microsoft.com/windows)
[![Registry](https://img.shields.io/badge/Edit-Registry-0078D6?style=flat-square&logo=windows)](https://learn.microsoft.com/ru-ru/windows/win32/sysinfo/registry)

**Автоматическое назначение высокого приоритета процессора для игр Valve в Windows.**

Этот репозиторий содержит `.reg` файлы, которые одним нажатием добавляют в реестр Windows настройку для запуска игр с **высоким приоритетом (High)**. Это позволяет игре получать больше процессорного времени, что может снизить количество микро-задержек (статтеров) и повысить стабильность FPS.

## 📦 Содержимое

| Файл | Назначение |
| :--- | :--- |
| `cs2_high.reg` | Устанавливает высокий приоритет для **Counter-Strike 2** (`cs2.exe`) |
| `deadlock_high.reg` | Устанавливает высокий приоритет для **Deadlock** (`deadlock.exe`) |

## 🚀 Быстрый старт

Всё, что нужно сделать — запустить `.reg` файл и подтвердить действие.

1.  Скачайте нужный `.reg` файл из репозитория.
2.  Нажмите на файл правой кнопкой мыши и выберите **"Объединить"** (или просто дважды кликните по нему).
3.  Нажмите **"Да"** во всплывающем окне контроля учётных записей (UAC) и **"Да"** в окне подтверждения редактора реестра.
4.  **Перезагрузите компьютер**, чтобы изменения вступили в силу.

Готово! После перезагрузки игра всегда будет запускаться с высоким приоритетом.

## 🛠️ Как проверить результат

1.  Запустите игру.
2.  Откройте **Диспетчер задач** (`Ctrl + Shift + Esc`).
3.  Перейдите на вкладку **«Подробности»**.
4.  Найдите процесс игры (`cs2.exe` или `deadlock.exe`).
5.  Нажмите на нём правой кнопкой мыши и наведите на пункт **«Задать приоритет»**.

Напротив пункта **«Высокий»** должна стоять отметка. ✅


## ❓ Частые вопросы (FAQ)

<details>
<summary><b>Почему не стоит использовать приоритет «Реального времени» (Realtime)?</b></summary>

Значение **Realtime (4)** может заставить Windows игнорировать важные системные процессы, что приведёт к зависанию мыши, клавиатуры и всей системы. Для игр **безопасен и оптимален именно «Высокий» (High)** приоритет.
</details>

<details>
<summary><b>Нужно ли запускать файл от имени администратора?</b></summary>

Обычно достаточно двойного клика. Если возникает ошибка доступа, нажмите на файл правой кнопкой мыши и выберите **«Запуск от имени администратора»**.
</details>

<details>
<summary><b>Изменения постоянны?</b></summary>

Да, они сохраняются в реестре Windows. Приоритет будет применяться при каждом запуске игры до тех пор, пока вы не удалите настройку.
</details>
